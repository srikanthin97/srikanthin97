*   **Tech Stack:** Python, Streamlit, SQLAlchemy, Pydantic, Plotly Express, LangGraph, Google Gemini API
*   **Key Capabilities:**

    *   **Autonomous Analyst Mode:** Uses a LangGraph state machine to autonomously plan sub-queries, run executions, and synthesize tabular data into Markdown executive insights reports.
